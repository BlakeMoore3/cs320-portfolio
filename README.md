# cs320-portfolio

This repository contains my work from CS 320, focusing on software testing and unit testing using Java and JUnit.

# How can i ensure that my code, program, or software is functional and secure?
I ensure my code is functional by writing unit tests that will cover both normal use cases and edge cases. Using JUnit, I tested each feature to confirm that it meets the requirements. To improve security, I validate inputs like ID length, null values, alnd required fields to prevent invalid data from being used.

# How do I interpret user needs and incorporate them into a program?
I interpret user needs by reviewing the requirements and breaking them into smaller features. An example would be creating and updating contacts becomes specific methods with rules. I then write tests based on those requirements to make sure the program behaves as expected. 

# How do I approach designing software?
I design software by separating responsibilities into different classes, like objects and services. This keeps the code organized and easier to maintain. I also design with testing in mind, making sure each part can be tested independently.
